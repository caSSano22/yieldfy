— OPTIMIZER CYCLE

Trigger. Score. Attest. Route.

01

Trigger

Chainlink Automation runs a keeper check every 6 hours. No cron job, no key sitting on a server.

02

Score

Pull live APY, TVL, and risk data from every connected venue. Hash the inputs so the result is reproducible.

03

Attest

Post the decision on-chain: winner, runner-up, expected extra yield in bps, and the input hash.

04

Route

If the extra yield beats the rebalance threshold, move funds up to the per-venue cap. Otherwise skip and wait.

— SCORING FORMULA

Five inputs. One winner.

Every venue gets scored on five numbers, weighted and added up. Highest score wins. The optimizer also estimates how much extra yield the switch is worth, so it only moves capital when the upside beats the gas and slippage. Stablecoins and stock tokens use different weightings picked at deposit.

Net APY

35% · Live yield after fees

Liquidity depth

25% · How much can move in and out

Protocol risk

20% · Audit status and TVL

Correlation buffer

15% · Not everything in one place

Rebalance cost

5% · Gas plus slippage estimate

Equity profile

Borrow demand, LP fees vs IL, volatility

Yieldfy Technical Specification

ERC-4626 Deterministic Vault Routing Standard

1. Smart Contract Architecture

Yieldfy employs standard ERC-4626 tokenized vault contracts deployed on Robinhood Chain. All capital routing is triggered via Chainlink Automation keepers querying the on-chain DeterministicOptimizer.sol module.

2. SDK Quickstart

// Install SDK
npm install @yieldfy/sdk

// Query top yield venue
import { YieldfyClient } from '@yieldfy/sdk';

const yieldfy = new YieldfyClient({ network: 'robinhood' });
const decision = await yieldfy.getBestVenue('AAPL');

console.log(`Routed Venue: ${decision.winnerVenue}`);
console.log(`Attestation Hash: ${decision.inputHash}`);

3. Scoring Weightings

  • Net APY (35%): Realized yield net of management fees.
  • Liquidity Depth (25%): Instantaneous depth for withdrawal guarantee.
  • Protocol Risk (20%): TVL, audit score, time-in-service.
  • Correlation Buffer (15%): Diversity factor across collateral pools.
  • Rebalance Cost (5%): Estimated gas overhead & swap slippage.
